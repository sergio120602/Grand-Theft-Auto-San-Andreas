------- Instructions:

Extract all files from the folder "(mod - to the game folder)" to your GTA SA folder (where the .exe is).
Can't be installed on ModLoader, as this mod needs to be loaded first.

-- Essentials Pack download: https://www.mixmods.com.br/2019/06/sa-essentials-pack/

The mod automatically updates the "CrashList.txt" file based on: https://github.com/JuniorDjjr/CrashInfo/tree/main/Lists
In the file "CrashInfo.ini" there are some language, update and connection options.
By default, it will update the "CrashList.txt" file once a day when starting the game.

You can test the mod by forcing it to crash, for example by renaming the "audio" folder.

The file "gta_sa.pdb" must have the same name as the .exe, only Hoodlum.
It shows the function names under the "Backtrace", which can help you understand the problem, but don't trust it.

Recommended to use together with SCRLog: https://www.mixmods.com.br/2020/09/scrlog.html
